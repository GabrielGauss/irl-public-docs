# Contributing

This repository contains public documentation for MacroPulse IRL. Contributions are welcome for:

- Typo and clarity fixes
- Additional language examples in the developer guide
- Exchange integration notes for venues not yet covered
- Translation of core docs

## What NOT to submit

- API keys, tokens, database credentials
- Customer-specific configurations or data
- Proprietary trading strategies or signals

## Process

1. Open an issue describing the change
2. Fork the repo, make the change, open a pull request
3. All contributions are reviewed before merge
