# Security Policy

If you discover a security vulnerability in the IRL Engine, MacroPulse platform, or any associated SDK, please report it responsibly.

**Do not open a public GitHub issue.**

Email **security@macropulse.live** with a description of the issue, steps to reproduce, and affected versions. You will receive an acknowledgement within 48 hours.

For SDK-specific issues, see:
- [irl-engine/SECURITY.md](https://github.com/GabrielGauss/irl-engine/blob/main/SECURITY.md)
- [irl-sdk-python/SECURITY.md](https://github.com/GabrielGauss/irl-sdk-python/blob/main/SECURITY.md)
- [irl-sdk-ts/SECURITY.md](https://github.com/GabrielGauss/irl-sdk-ts/blob/main/SECURITY.md)
